About this Website

Mobowiki lets you know everything you would like to know about any smartphone. It provides Pros, Cons, Complete Specification followed by a verdict. There is a comment section, which allows a user to post any comment as well as view the comments made by different users on a particular smartphone. Mobowiki also keeps a track of your favourite smartphones in the wishlist section which can be found on the top right corner.
